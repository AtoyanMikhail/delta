
## Run Locally

Clone the project

```bash
  git clone https://github.com/AtoyanMikhail/delta.git
```

Go to the project directory

```bash
  cd delta
```

Install go 1.22 or higher with

```bash
sudo snap install go 1.22 --classic
```

Install dependencies 

```bash
go mod tidy
```

Build the project with
```bash
./build.sh
```
